# 🎙️ VoxCPM2 Realtime Mic (보이스 클로닝 & 마이크 송출)

VoxCPM2를 활용하여 텍스트를 고음질 음성으로 변환하거나 목소리를 복제하고, 이를 실시간 마이크 입력으로 쏴주는 오픈소스 웹 UI입니다.

## 🚀 시작하기 (Windows)

1. 이 저장소를 다운로드(Clone) 합니다.
2. 폴더 안에 있는 `run.bat` 파일을 더블클릭합니다.
3. 모든 설치가 자동으로 진행되며 인터넷 브라우저에 UI가 열립니다.

## 🎧 실시간 마이크 송출 세팅 (중요!)

디스코드, 줌, 인게임 보이스 등에서 생성된 목소리를 내 마이크처럼 사용하려면 **가상 오디오 케이블**이 필요합니다.

1. [VB-Audio Virtual Cable](https://vb-audio.com/Cable/) 에 접속하여 무료 드라이버를 설치하고 재부팅합니다.
2. 디스코드나 게임의 **마이크(입력) 장치**를 `CABLE Output (VB-Audio Virtual Cable)`으로 변경합니다.
3. 본 프로그램(웹 UI)의 **오디오 송출 장치** 목록에서 `CABLE Input`이 포함된 장치를 선택합니다.
4. 이제 텍스트를 입력하고 버튼을 누르면 상대방에게 목소리가 바로 들립니다!

## 💡 주요 기능
* **보이스 디자인:** 텍스트 프롬프트만으로 나만의 목소리 톤을 만들어냅니다.
* **보이스 클로닝:** 3~5초의 짧은 오디오 파일만 업로드하면 해당 목소리와 억양을 그대로 복제하여 읽어줍니다.
