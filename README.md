# 🎙️ VoxCPM2 Realtime Translator & Mic (실시간 음성 번역 송출기)

VoxCPM2와 OpenAI Whisper 엔진을 결합하여, 사용자의 마이크 입력 음성을 전 세계 100여 개 이상의 언어로 실시간 인식 및 번역하고, 복제된 목소리(또는 디자인된 목소리)로 가상 마이크에 즉시 송출해 주는 데스크톱 GUI 애플리케이션입니다.

디스코드, 줌(Zoom), 인게임 보이스챗 등에서 원어민처럼 자연스럽게 다국어 음성 소통을 하고 싶을 때 매우 유용합니다.

---

## ✨ 주요 기능

* **눌러서 말하기 (Push-to-Talk) 지원:** 데스크톱 프로그램의 버튼을 꾹 누르는 동안에만 음성을 녹음하여 연산 지연 및 오작동을 최소화합니다.
* **전 세계 모든 언어 동적 지원:** `deep-translator` 라이브러리와 연동되어 전 세계 100개 이상의 언어를 자동으로 불러와 양방향 번역을 수행합니다.
* **보이스 디자인 (Voice Design):** 간단한 영문 텍스트 프롬프트 설명만으로 원하는 목소리 톤을 생성합니다.
* **보이스 클로닝 (Voice Cloning):** 3~5초 분량의 단일 오디오 파일(.wav) 경로만 지정하면 해당 목소리의 성향과 억양을 그대로 복제하여 송출합니다.
* **비비동기 스레드 연산:** 딥러닝 추론 연산 중에도 프로그램 창이 멈추거나 튕기지 않고 실시간 처리 로그를 안전하게 출력합니다.

## 🛠️ 필수 사전 설치 (Prerequisites) - 중요!

초보자분들이 가장 많이 겪는 오류를 방지하기 위해 프로그램 실행 전 아래 두 가지 설정을 반드시 완료해 주세요.

### 1. 가상 오디오 케이블 설치
생성된 AI 음성을 내 실제 마이크 입력처럼 디스코드나 게임에 쏴주기 위해 필요합니다.
1. [VB-Audio Virtual Cable 공식 홈페이지](https://vb-audio.com/Cable/)에 접속하여 무료 드라이버를 다운로드하고 설치합니다.
2. 컴퓨터를 재부팅합니다.
3. 디스코드나 사용하실 게임의 **마이크(입력) 장치** 설정을 `CABLE Output (VB-Audio Virtual Cable)`으로 변경합니다.

### 2. FFmpeg 설치 및 환경 변수 등록
음성 인식 엔진(Whisper)이 마이크 입력을 정상적으로 자르고 처리하기 위해 필수적인 소프트웨어입니다.
1. [FFmpeg 공식 사이트](https://ffmpeg.org/download.html) 또는 패키지 관리자를 통해 Windows용 FFmpeg를 설치합니다.
2. `ffmpeg.exe`가 있는 폴더 경로를 윈도우 **시스템 환경 변수(PATH)**에 반드시 등록해야 합니다. (등록하지 않으면 실행 시 `FileNotFoundError`가 발생합니다.)

## 🚀 시작하기 (Windows)

1. 본 저장소를 다운로드(Clone) 하거나 압축 파일로 풀어줍니다.
2. 폴더 내부에 있는 `run.bat` 파일을 마우스 우클릭하여 **관리자 권한으로 실행**하거나 더블 클릭합니다.
3. 프로그램이 컴퓨터 환경을 자동으로 진단합니다:
   * 시스템 내에 설치된 파이썬 버전 중 **호환성이 검증된 3.12 이하 버전**을 자동으로 찾아 가상환경(`venv`)을 빌드합니다.
   * NVIDIA 그래픽 카드 및 CUDA 드라이버 존재 여부를 감지하여 시스템에 맞는 PyTorch 버전을 추천합니다.
4. 안내에 따라 하드웨어 환경(1번 CPU / 2번 GPU)을 선택하면 필요한 모든 라이브러리가 자동으로 설치된 후 데스크톱 프로그램이 실행됩니다.

> 💡 **최초 실행 시 주의사항:** 프로그램을 처음 켜고 마이크 버튼을 떼는 순간, VoxCPM2 모델과 Whisper 모델 가중치(수 GB)를 인터넷에서 자동으로 다운로드합니다. 네트워크 환경에 따라 첫 연산은 수 분이 소요될 수 있으므로 검은색 터미널 창을 끄지 말고 잠시만 기다려 주세요. 두 번째 실행부터는 실시간으로 즉시 작동합니다.

## 🎛️ 프로그램 사용 방법

1. **오디오 설정:** 상단 드롭다운 메뉴에서 가상 오디오 케이블 장치(`CABLE Input`이 포함된 이름)를 선택합니다. (단순히 내 스피커로 테스트해보고 싶다면 `장치 선택 안함 (미리듣기만)`으로 두셔도 됩니다.)
2. **목소리 프로필 설정:**
   * **보이스 디자인:** 영문으로 목소리 성향을 적습니다. (예: `A calm and clear male voice`, `A young woman, gentle and sweet voice`)
   * **보이스 클로닝:** 원하는 목소리가 담긴 3~5초짜리 깨끗한 독백 `.wav` 파일의 전체 경로를 복사하여 입력창에 넣고 라디오 버튼을 전환합니다.
3. **언어 설정:** '내 언어(입력)'와 '번역 언어(출력)'를 전 세계 언어 목록에서 자유롭게 선택합니다.
4. **송출하기:** 하단의 빨간색 버튼을 **마우스로 꾹 누른 상태에서** 마이크에 대고 말을 합니다. 말씀이 끝나면 마우스 버튼을 뗍니다. 시스템이 자동으로 번역하여 상대방에게 내보냅니다.
